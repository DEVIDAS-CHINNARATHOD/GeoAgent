# GeoAgent

## Project Description
GeoAgent is an agentic AI-powered local tourism assistant that generates personalized travel plans using multi-agent reasoning, real-time data, and retrieval-based knowledge systems.

## Project Overview
GeoAgent is a multi-agent AI system designed to assist users in planning local travel experiences. It combines retrieval-augmented generation (RAG), real-time APIs, and intelligent agents to deliver personalized, context-aware itineraries.

## What We Are Building
We are developing a production-ready agentic AI system that:
- Generates structured travel itineraries
- Recommends nearby places based on location
- Adapts plans using real-time weather data
- Provides safety and travel risk insights
- Personalizes recommendations based on user preferences
- Exports travel plans as downloadable files

## Key System Components
- Planner Agent (task decomposition)
- Explorer Agent (RAG-based retrieval)
- Nearby Agent (maps integration)
- Weather Agent (real-time data)
- Safety Agent (risk analysis)
- Aggregator Agent (final response generation)
